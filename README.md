# teste_dotz

Este é um projeto React chamado teste_dotz

## Executando o Projeto

1. Clone o repositório:

    ```bash
    git clone https://github.com/puegue1999/teste-dotz.git
    ```

2. Navegue até o diretório do projeto:

    ```bash
    cd teste_dotz
    ```

3. Intale as dependências:

    ```bash
    npm install
    ```

4. Execute o projeto:

    ```bash
    npm start
    ```
